# Troubleshooting

**Read `iteration-log.md` first every conversation.** It records what happened each test run so you can pick up where we left off without the user re-describing symptoms.

This folder is the reference for diagnosing cross-platform differences between:

- `ActivitiesApp.Api`
- `ActivitiesApp.Web` (Blazor)
- `ActivitiesApp` (MAUI)

Files:

- `iteration-log.md`: **start here** — running record of each test iteration: symptoms, root cause, fixes, and what to verify next
- `discover-and-images.md`: reference for activity discover flow, category caching, image delivery, and log line prefixes to search for
